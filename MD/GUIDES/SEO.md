# SEO Best Practices – Stand Mai 2026

Diese Dokumentation beschreibt die kritischen SEO-Faktoren für das Jahr 2026, mit Fokus auf Generative Suche (AI), Benutzererfahrung und technische Exzellenz.

## 1. Generative Search Optimization (GEO/ASO)
Die klassische Suche wurde größtenteils durch KI-generierte Antworten (SGE - Search Generative Experience) abgelöst.
- **Answer Engine Optimization (AEO):** Inhalte müssen so strukturiert sein, dass sie von KI-Modellen (Google Gemini, OpenAI Search) leicht extrahiert und als Zitat verwendet werden können.
- **Strukturierte Daten (JSON-LD):** Vollständige Implementierung von Schema.org (Organization, Service, VideoObject, FAQ) ist Pflicht, um als verifizierte Quelle in KI-Antworten zu erscheinen.
- **Citations & Branding:** Erwähnungen auf autoritären Plattformen sind wichtiger als reine Backlinks.

## 2. E-E-A-T 2.0 (Experience focus)
In einer Welt voller KI-Content belohnt Google echte menschliche Erfahrung.
- **First-Hand Experience:** Zeige echte Projekte, Behind-the-Scenes Videos und individuelle Fallstudien.
- **Author Authority:** Jede Seite sollte einem Experten zugeordnet sein (biografische Daten, soziale Profile).
- **Transparency:** Klare Angaben zu Impressum, Datenschutz und Kontakt (Vertrauenssignale).

## 3. Technical SEO & Core Web Vitals
Geschwindigkeit ist kein "Bonus" mehr, sondern eine Grundvoraussetzung.
- **Interaction to Next Paint (INP):** Optimierung der Reaktionszeit bei Benutzerinteraktionen auf unter 200ms.
- **Mobile-First Indexing:** Die Seite muss primär auf mobilen High-End Displays (OLED, hohe Bildwiederholraten) perfekt funktionieren.
- **Accessibility (WCAG 2.2):** Barrierefreiheit ist ein direkter Rankingfaktor. Screenreader-Kompatibilität und Farbkontraste müssen stimmen.

## 4. Content & Semantic Search
Weg von Keywords, hin zu Entitäten und Intent.
- **Topic Clusters:** Anstatt für einzelne Keywords zu optimieren, muss eine gesamte Themenwelt (z.B. "High-End Videoproduktion") abgedeckt werden.
- **Conversational Queries:** Optimierung auf natürliche Sprache ("Wie erstelle ich einen Imagefilm für Recruiting?").
- **Visual SEO:** Hochwertige WebP/AVIF Bilder mit aussagekräftigen Alt-Texten und Video-Transkripten.

## 5. Pflicht-Checkliste vor Go-Live

- [ ] Canonical Tag gesetzt
- [ ] JSON-LD Schema (Organization / ProfessionalService) vorhanden
- [ ] H1 eindeutig, keyword-reich, 1× pro Seite
- [ ] Alt-Tags auf allen Bildern (beschreibend, max 125 Zeichen)
- [ ] OpenGraph + Twitter Cards konfiguriert
- [ ] Sitemap.xml + robots.txt im Root
- [ ] INP < 200ms
- [ ] Mobile-First überprüft
